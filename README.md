## Overview
This repository contains a personal portfolio website for Khant Lin Oo, a Data Analyst specializing in Power BI, SQL, and Python. The site presents a polished one-page experience that highlights analytical projects, technical skills, and ways to get in touch.

## Project Structure
- index.html — main portfolio page structure
- styles.css — visual styling, layout, and responsive behavior
- script.js — interactive behavior such as theme toggling, animations, and project filtering
- projects/ — project assets and related files for each case study
  - python_project/ — Python-based movie analysis reports
  - powerbi_project/ — Power BI report and dataset files
  - sql_sales_analysis/ — SQL project files and documentation
  - Khant_Lin_Oo_Data_Analyst_Resume.pdf — downloadable resume

## What This Project Covers
The portfolio showcases:
- A responsive one-page layout with hero, about, work, skills, and contact sections
- Filterable project cards for Python, SQL, and Power BI work
- A light/dark theme toggle and animated UI elements
- Links to project case studies and external GitHub resources

## Requirements
- A modern web browser
- Optional: a simple local server for previewing the site more accurately

## How to Use
1. Clone or download this repository.
2. Open index.html in your browser, or run a local server from the project folder.
3. Browse the portfolio and click through the featured projects.
