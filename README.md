# Khant Lin Oo Portfolio

This repository contains a personal portfolio website for Khant Lin Oo, a Data Analyst specializing in Power BI, SQL, and Python.

## What this site includes

- A responsive one-page portfolio layout
- A hero section with profile information and project highlights
- An About section describing the analyst’s background and interests
- A case-study section for three data projects
- A skills section showing core tools and proficiency
- A contact section with social links and email
- A light/dark theme toggle and animated UI elements

## Project highlights

The portfolio showcases:

- Movie industry performance analysis using Python and Jupyter
- SQL-based employee compensation and workforce analysis
- Power BI sales dashboard built from Adidas US sales data

## Main files

- index.html — main portfolio page structure
- styles.css — visual styling, layout, and responsive behavior
- script.js — interactive behavior such as theme toggling and animated counters
- projects/ — project assets and related files for each case study

## Run locally

Open index.html in a browser, or serve the folder with a simple local server if you want a more realistic preview.
