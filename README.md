# Portfolio Website

This is a static portfolio for Khant Lin Oo.

## GitHub Pages

To publish this repository with GitHub Pages:

1. Push the repo to GitHub.
2. In repository settings, enable Pages from the `main` branch (or `gh-pages` branch) and use the root folder.
3. Wait a few minutes and visit `https://<username>.github.io/<repo-name>/`.

## Notes

- The site is entirely static and uses `index.html`, `styles.css`, and `script.js`.
- `lucide` icons are loaded from `https://unpkg.com/lucide@latest/dist/umd/lucide.js`.
- The `projects/` folder contains linked project assets.
