# Power BI Sales Analysis Project

## Overview
This project contains a Power BI dashboard and supporting data files for an Adidas US sales analysis. The report visualizes sales performance and business insights using a Power BI .pbix file and a preview HTML version.

## Files
- Project Assignment KLO.pbix - Main Power BI report file containing the dashboard and visualizations
- Project_Assignment_KLO_preview.html - HTML preview of the report for quick viewing in a browser
- Adidas US Sales Datasets.xlsx - Source dataset used to build the report

## What This Project Covers
The dashboard focuses on analyzing sales data, including:
- Sales trends over time
- Product and regional performance
- Revenue and order insights
- Business-oriented visual summaries for decision-making

## Requirements
To use the project fully, you need:
- Microsoft Power BI Desktop
- The Excel dataset file
- A browser to view the HTML preview if needed

## How to Use
1. Open the Power BI file in Power BI Desktop.
2. Connect or refresh the data using the included Excel dataset.
3. Explore the dashboard visuals and report pages.
4. Use the HTML preview for a quick overview of the project.

## Notes
- The .pbix file is the primary working report.
- The Excel file serves as the data source for the dashboard.
- The HTML preview is intended for easy sharing and quick viewing.
